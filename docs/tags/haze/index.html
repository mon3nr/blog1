    小松川公園ハゼ釣行その１

    この記事では江戸川区大島小松川公園の旧中川の釣り場を紹介します。 ハゼ釣りで有名なポイントで、隣の荒川に比べて穏やかな潮流です。 簡単な仕掛けでも釣れる人気の釣り場です。 しかし、スレている魚が多いので、 ハゼが溜まっている場所じゃないと、アタリが出ずらい。 アタリが出ても、掛からない、バレてしまう。 といったケースが多くありました。 経験からどうすれば釣れるか、お勧めの釣り方を紹介したいと思います。 内容は初心者向けです。 ポイントについて 旧中川について 護岸、木の杭について 対象魚のハゼについて ハゼの成長にアワセた釣り方 6月から9月は落とし込みのミャク釣り 10月から翌年1月は天秤仕掛けのちょい投げ ボラについて 6月-9月のシーズンのミャク釣り お勧めのタックル 釣り方 まとめ ポイントについて 旧中川について 旧中川は本流の荒川(荒川放水路)から分かれた河川で、南側近くにある荒川ロックゲートで再び合流します。 西側はに小名木川があります。 小名木川は東西に流れる運河で、その先は隅田川とつながります。 荒川の本流に比べると潮流は穏やかで、仕掛けが流される心配は殆どありません。 水陸両用バスの船着き場があり、付近の船道は水深が2～3m と深くなります。 岸部にごろた石があり、その先は砂泥となります。 砂泥帯に障害物などはなく、根掛かりすることはまずありません。 小名木川合流付近で岸まで80-90mと広く、東側は高台の公園となっているため、湖畔のようなイメージのある綺麗な釣り場です。 主なポイントは下図の西側のロックゲート側、中川大橋付近側の2か所になります。 護岸、木の杭について 岸部のごろた石の際に木の杭とロープが張られています。 これは、ボートの侵入を防ぐために作られたもので、ロープまでは20cmほどの浅さですが、 その先はカケアガリとなり、1m 前後と急に深くなります。 カケアガリは藻などの魚の餌が溜まりやすく、チチブなどのハゼの隠れ家になっています。 このロープの先のカケアガリに落とすことが重要なポイントになります。 ロープまでは2.5m～3mあり、3m 前後の竿の方がやり取りがしやすく、お勧めです。 なお、投げ釣りは禁止となっているため、お勧めの釣り方は、仕掛けを落とす落とし込み（ミャク釣り）又は、アンダーキャストでのちょい投げになります。 対象魚のハゼについて ハゼの成長にアワセた釣り方 ハゼの一生は1年で、冬から春先にかけて産卵、孵化し、約1年かけて成長し、産卵後に死にます。 越冬して2年目を迎えるハゼもおり、これらは落ちハゼと呼ばれ、一周り大きいハゼが釣れます。 ハゼの成長に合わせて仕掛け、釣り方を変えるのが重要で、主に以下の2パターンに分かれます。 ① 6月から9月は落とし込みのミャク釣り 梅雨時を迎えるとデキハゼと呼ばれ、やっと針に掛かる大きさまで成長します。 この時期のサイズは5～7cmの小型で徐々に成長します。 8月になると 8～10cm が中心となります。 この時期のハゼはナツハゼと呼ばれ、一番釣れる時期になります。 いずれも、水深１ｍもないごく浅いカケアガリを好み、前述の木の杭の先のカケアガリに仕掛けを落とす、落とし込みのミャク釣りが適しています。...
