    東京ウナギ釣りポイント調査編

    October 24, 2021

    今夏からウナギ釣りを始めました。 ブッコミ釣りの仕掛けを投げて待つだけの、テクニックはあまりいらない 比較的簡単な釣り方でしたが、都心の身近な河川で釣果を得ることができました。 しかし、釣り場を探すのはとても大変でした。 東京は開削した河川が多く流れており、釣り場 は上流から下流まで広範囲にわたり、何処で釣れるか／釣るかを探す必要ありました。 SNSの釣果情報からポイントを探して、現場調査（釣行）を繰り返しました。 その結果、9月に 60cm の良型 2 匹を掛けることができました。

    本記事ではSNS釣果情報からウナギ釣りの分析をした結果をまとめます。 実釣記録については改めて報告します。

    ウナギ釣りポイント探し#

    釣り専用SNSサイト Anglers などの 釣果情報から、東京近郊の河川で広範囲にウナギが釣れています。 これらの河川は、開削された人工河川が多く、それぞれの河川で護岸の形状に特徴があります。 これら河川で狙うポイントは、 以下の「ウナギの隠れることができる障害物が多い場所を探る」 ことです。

    • アシ原、水草などが水生植物が生えている場所
    • 橋脚、水門、船着き場などの付近で潮流に変化が出やすい場所
    • ゴロタ石などのガレ場

    Anglers 釣果情報からウナギが釣れるポイントを検索し、Googleマップ で付近の河川の形状を探りました。

    また、ブログなどにもウナギ釣りに関する情報が多くあり、 その中でも特に記事が豊富でまとまったサイトとして、 関東ウナギ釣り掲示板 がありました。

    今回、本掲示板からウナギが「釣れる」ポイントを分析した結果を記します。

    掲示板記事をCSVに変換する#

    hazebbs.com の釣り掲示板には以下の4つの関東周辺のウナギ釣りの掲示板がありました。

    • 関東ウナギ釣り(その2)

    • 関東ウナギ釣り(その3)

    • 東京のウナギ釣り場

    これら記事を自前の Python スクリプト で CSV に変換し、テキスト分析ツール KH Coder に読み込ませて分析しました。 Python スクリプト、 KH Coder の利用方法については以下の Readme.md を参照してください。

    東京ウナギ釣り掲示板記事抽出 [https://github.com/mon3nr/unasearch]

    変換した CSV を Excel に読み込ませて、記事の件数をカウントした結果は以下になります。 6月～9月にかけての記事が集中しており、その期間がウナギ釣りのシーズンとなり、 冬場はシーズンオフのようです。

    抽出語の分析#

    初めに抽出語の件数の分析をしました。KH Coder に CSV ファイルを読み込ませ、 抽出語件数のリストを作成しました。 「釣れる」が抽出件数のランキング1位で、リンクをクリックをすると「釣れる」 の前後の文章リストが表示されます。 リストを順に選択して文章を表示し、「釣れる」に関するキーワードでいつ、どこで、 どうやって釣れるのかを把握しました。

    「釣れる」、「釣れない」ポイントの分析#

    コーディングルールという関連するキーワードのグループを定義しました。 本ルールは明示的にどういった記事が含まれるかのコンセプトを示すもので、 コンセプトに関連するキーワードを定義します。 例えば、「釣れる」、「釣れない」のコンセプトを以下のとおり定義しました。

    コンセプト キーワード　
    釣れる 釣れる, 釣れた, 釣れます, 釣れてます, 当たり, 好調, キープ, 太い, 大物
    釣れない 渋い, 釣れません, 釣れない, 坊主, ボウズ, オデコ, ダメ, だめ, 駄目, アタリなし, アタリすらない, 不調

    定義したコーディングルールは、 Rule.txt となり、 定義したコンセプトと各河川のクロス集計結果が以下になります。

    赤色が濃いプロットがコンセプトとの関連が強い記事の出現率が高く、 四角の面積は各ポイントでコンセプトが出現した割合を示しています。 色の濃く、面積の大きいものが釣れるポイントの目安となり、 新中川、中川、平和橋、利根川がその可能性が高いと考えます。

    「釣れる」関連語の分析#

    KH Coder メニューから 関連語検索 を選択し「釣れる」に関連する記事を抽出し、 抽出したリストの共起ネットを作成したものが下図のとおりです。

    関連の強い語のネットワークをあらわしたもので、理解しにくい部分は多々ありますが、 大まかに以下のポイントで、特徴のある語との関連があるようです。

    • 平和橋（中川にある橋）
    • 新中川、江戸川は同じ傾向
    • 中川、キープサイズ

    どのような関連があるのかの詳細は記事の中身を見る必要がありますが、 大まかに上記がウナギ釣りに適したポイントだと考えます。 次節で上記ポイントの現地調査結果を記します。

    現地調査#

    中川、平和橋#

    最も「釣れる」関連が強いポイントになります。 広い河川で幅は、143mあります。 コンクリート護岸で整備されており、とても綺麗な釣り場です。 一面テラスの柵が敷かれており、竿受けは不要で、敷地も広いため釣行はしやすいかと思います。 ただ、かなり整備された河川で、ガレ場など自然物はありませんでした。 狙えるポイントは橋脚付近など限定されると思います。 釣り人は橋脚付近に2組見かけましたが、広い釣り場のため、釣り座の確保の心配は無さそうです。

    護岸の形状は垂直式のコンクリート護岸で、写真の通りオーバーバンクの傾斜があるため、 ヘチ釣りにはコツがいりそうです。 周辺一帯は砂泥帯で根掛かりの心配はなさそうです。

    新中川#

    こちらは開削された人工河川で、幅は75mの比較的小さめの河川です。 船着き場があり、ウナギの隠れ家になりそうな障害物がありました。 柵があるため直接投げることはできませんが、その周辺がねらい目になりそうです。 一面、テラスの柵があり、竿受けは不要です。 垂直式のコンクリート護岸で、底は砂泥帯できつい根はなさそうです。 釣り人は少なく、１組見かけました。

    また、アシ原の植生護岸があり、餌が溜まりやすいポイントで、こちらも狙い目です。 アシ原はコンクリートブロックに囲まれており、コンクリートで隔てられているので 付近に投げても根掛かりの心配はなさそうです。

    江戸川、なぎさ公園#

    江戸川、旧江戸川は下流から上流まで広範囲になり、釣り場は多数ありますが、 近所のなぎさ公園を調査しました。 ゴロタ石のガレ場が続く河川となり、ゴロタ石の先は砂泥帯の様です。 狙い目はその際に落とすのが良いでしょう 。

    公園入口の船着き場周辺にはテラスがあり、写真と通りかなり多くの釣り人がいました。 テラス側に釣り座を確保できない場合は、竿受けを持参した方が良いかと思います。 アシ原などの水草も点在しており、ウナギの隠れ家となる障害物が多くありました。 河口に近いため、ミミズ餌だと海水ですぐに弱ってしまうため、イソメ類の方が適しています。

    まとめ#

    以上、東京のウナギ釣りポイント調査になります。 今回、 KH Coder を用いた SNS 釣果情報の分析をしました。

    あまり専門的な知識がなく高度な分析はできませんでしたが、 ツールを用いることで簡単な操作で分析ができました。 釣り場が広範囲で、何処がお勧めなのかの目安を知るには良いアプローチだと思います。

    本執筆時期は10月となりウナギ釣りにはシーズンオフとなるため、 来春から実釣を再開し、実釣記録をまとめたいと思います。
